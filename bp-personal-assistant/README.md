# 🧠 مُعين — المساعد الشخصي لباهر (WhatsApp × Claude × Notion)

> مساعد شخصي **لباهر وحده** عبر واتساب، مربوط بـ Claude ونوشن والجيميل والكالندر — يريّحك من النسيان والتسويف: يسجّل مهامك، يرتّب مواعيدك، يوثّق اجتماعاتك، يضيف عقاراتك وجهات اتصالك، ويحسب مصاريفك واشتراكاتك. بديل أرخص وأقوى من [notis.ai](https://notis.ai) لأنه مبني على بنيتك أنت (n8n + حساباتك).

## الفكرة

طبقة **شخصية** فوق منظومة Business Partner القائمة (n8n + Claude + Notion). منفصلة تماماً عن إيجنتات العملاء والشركة. تكلّم رقم واتساب واحد، والمساعد (اسمه «مُعين») يرد عليك بالعربي كإنسان، ويكتب في ٧ قواعد نوشن خاصة بك.

```
واتساب (باهر فقط)  →  Claude (مُعين)  →  ┌─ Notion MCP (٧ قواعد)
   owner-only                            ├─ Gmail (قراءة/تنظيف)
   + ذاكرة المحادثة                       └─ Google Calendar (مواعيد)
   →  رد على واتساب
```

## ماذا يفعل مُعين

| ترسل له… | يسوّي… | يكتب في… |
|---|---|---|
| «ذكّرني أراجع قوى بكرا الساعة ١٠» | مهمة + حدث كالندر | المهام + Calendar |
| محضر اجتماع | اجتماع + نقاط عمل كمهام | الاجتماعات + المهام |
| صورة/وصف عقار (نوع+سعر+موقع) | يضيف العقار | العقارات |
| «فلان 05xxxxxxxx مهتم بعقار» | جهة اتصال + ربط المصلحة | جهات الاتصال |
| رسالة تنبيه بنكي / إيميل فاتورة | مصروف مصنّف بالحساب الصحيح | المصاريف (+ الاشتراكات لو متكرر) |
| كود OTP | يخزّنه بأمان ويعطيك إياه | الوارد والرموز |
| «نظّف إيميلي» / «ايش الجديد» | يقرأ ويلخّص ويحوّل للمهام/المصاريف | Gmail → Notion |
| أي سؤال عن ملفاتك | يبحث في نوشن ويجيب | — |

## خطوات التشغيل (٥ دقائق)

1. افتح الـ workflow: <https://businesspartnerai.app.n8n.cloud/workflow/YgRwn40v1CqsscJw>
2. عدّل عقدة **Owner Only (Baher)** → ضع رقم واتساب باهر بدل `REPLACE_WITH_BAHER_WHATSAPP_NUMBER` (صيغة `9665xxxxxxxx`، بدون + أو أصفار بادئة).
3. تأكد أن الاعتمادات مربوطة في العقد: WhatsApp Trigger، WhatsApp send، Anthropic، Notion MCP، Gmail، Google Calendar (كلها موجودة مسبقاً — فقط اختر الصحيح إن طُلب).
4. فعّل الـ workflow (Active)، وتأكد أن رقم WhatsApp Cloud يوجّه الـ webhook لهذا الـ workflow.
5. أرسل من جوالك: «ذكّرني أتصل على الأستاذ أحمد الساعة ٥» — لازم يرد ويسجّل المهمة.

التفاصيل الكاملة في [`setup/whatsapp-owner-setup.md`](setup/whatsapp-owner-setup.md).

## المرحلة الثانية (مصمّمة، تُبنى عند الطلب)

- **سحب OTP/SMS/iMessage من الآيفون تلقائياً** عبر اختصار iOS → Webhook — الوصفة في [`setup/ios-shortcut-otp-sms.md`](setup/ios-shortcut-otp-sms.md).
- **مسح المصاريف والاشتراكات من الإيميل تلقائياً** (شخصي + شركة) بجدولة يومية — التصميم في [`setup/expenses-and-subscriptions.md`](setup/expenses-and-subscriptions.md).
- **إضافات سهلة للـ workflow:** إيميل الشركة كأداة Gmail ثانية، Google Drive Tool للملفات، قراءة الكالندر (getAll) لترتيب اليوم.

> ملاحظة أمان: قاعدة «الوارد والرموز» تحوي أكواد OTP حسّاسة — لا تُشارك الصفحة، ومُعين لا يرسل أي OTP خارج محادثة باهر.
