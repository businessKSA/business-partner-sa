# 📲 سحب OTP / SMS / iMessage من الآيفون → نوشن (المرحلة الثانية)

## لماذا اختصار iOS؟
آبل لا توفّر API مباشر لقراءة الرسائل النصية أو iMessage لأسباب خصوصية. الطريقة العملية الوحيدة الموثوقة: **اختصار (Shortcut) + أتمتة (Automation)** على جوالك، يعيد توجيه أي رسالة تصلك إلى Webhook في n8n، ومُعين ينظّفها ويسجّلها.

## المعمارية
```
رسالة SMS/OTP تصل الآيفون
   → Automation (When I get a Message)
   → Shortcut: Get Contents of URL (POST) → n8n Webhook
   → Claude ينظّف/يصنّف (OTP؟ بنك؟ اشتراك؟)
   → Notion: الوارد والرموز (+ المصاريف/الاشتراكات عند اللزوم)
```

## أ) جانب n8n (يُبنى عند الطلب)
Webhook `POST /baher-otp-in` يستقبل:
```json
{ "sender": "STC", "body": "رمز التحقق 483920 ...", "received_at": "2026-07-03T10:00:00Z" }
```
ثم AI Agent (Claude) يصنّف:
- **OTP** → `الوارد والرموز`: النوع=OTP، الكود=المستخرج، المُرسِل، النص، ينتهي=+10 دقائق.
- **تنبيه بنكي بمبلغ** → `المصاريف` (مصدر=SMS بنك) + `الوارد والرموز`.
- **تجديد/فاتورة اشتراك** → `الاشتراكات` (التجديد القادم) + `المصاريف`.
- غير ذلك → `الوارد والرموز` النوع=SMS.

> هذا الـ workflow جاهز للبناء بنفس نمط الـ workflow الرئيسي — قل «ابنِ لي OTP ingest» وأبنيه وأربطه.

## ب) جانب الآيفون (تسوّيه أنت مرة واحدة)

### اختصار «أرسل للمساعد»
1. تطبيق **Shortcuts** → **+** → اسمه `أرسل للمساعد`.
2. أضف إجراء **Get Contents of URL**:
   - URL: رابط الـ webhook (أعطيك إياه بعد بناء الـ workflow، مثل `https://businesspartnerai.app.n8n.cloud/webhook/baher-otp-in`).
   - Method: **POST** · Request Body: **JSON**.
   - حقول JSON:
     - `sender` = المتغيّر **Sender** (من الأتمتة)
     - `body` = المتغيّر **Content** (نص الرسالة)
     - `received_at` = **Current Date** (ISO 8601)
3. (اختياري) أضف **Show Notification** «أُرسل للمساعد ✅».

### الأتمتة (تشغيل تلقائي)
1. **Shortcuts → Automation → +→ Create Personal Automation**.
2. اختر **Message** → **Message Contains** = (اتركه عاماً، أو حدّد كلمات مثل «رمز/OTP/عملية شراء/خصم»).
3. **Run** → اختر اختصار `أرسل للمساعد`.
4. أطفئ **Ask Before Running** ليشتغل صامتاً.

> ملاحظة: أتمتة الرسائل على iOS قد تتطلب تأكيداً حسب إصدار النظام؛ إن لم تسمح بالتشغيل الصامت الكامل، تبقى بضغطة واحدة على الإشعار.

### بديل أبسط (بدون أتمتة)
شارك أي رسالة من تطبيق **Messages** عبر **Share → أرسل للمساعد**، أو ببساطة **أعد توجيه الرسالة كنص إلى رقم واتساب مُعين** — وسيصنّفها ويسجّلها فوراً (يعمل اليوم بدون أي إعداد إضافي).

## الخصوصية
- الأكواد تُخزَّن في قاعدة «الوارد والرموز» الخاصة فقط.
- مُعين لا يعيد إرسال أي OTP لأي جهة — يعرضه لك في محادثتك فقط.
- يُفضّل تفعيل حقل `ينتهي` وحذف/أرشفة الأكواد القديمة دورياً.
