# Facebook Playbook — Farah / Business Partner
**فيسبوك**

Reference sheet for Farah when producing Facebook content.

## OBJECTIVE
Reach the older SME-owner / office-manager segment that still lives on Facebook groups and pages; drive traffic to businesspartner.sa.

## FORMATS & SIZES
- Feed post: 1200×630 (link-post friendly) or 1080×1080 square.
- Prefer native text + image over external link when possible (better organic reach), with the link in the first comment when a click-through matters.

## CADENCE (v1 target)
3 posts/week. Cross-post from Instagram feed content but re-write the caption longer-form — Facebook audience reads more before scrolling.

## PILLAR MIX
Educational 30% · Risk/Deadlines 20% · Product/Offer 25% · Proof & Trust 20% · Behind-the-scenes 5%.

## TONE
Slightly more explanatory than Instagram — can run 4–6 lines, use full sentences, avoid excessive hashtags (2–5 max, Facebook hashtags carry less weight than IG).

## CAPTION TEMPLATE
```
[Hook / question]

[Explanation — 3-5 sentences, plain language]

[CTA: "اتصل بنا" / "زور الموقع" / رابط]

[2-5 hashtags]
```

## FORBIDDEN
- No boosted/paid ad claims without an approved budget on file.
- No political or religious commentary, even tangential.
- No client names/photos without consent.
