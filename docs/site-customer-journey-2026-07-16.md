## طبقة الذكاء — تمتد تحت كل مرحلة

زر المساعد العائم موجود على كل صفحة؛ لا ينتظر الزائر مرحلة معيّنة. التدفّق:

```
الزائر/العميل → المساعد العائم · /voice → خالد (رئيس الأركان)
→ 11 وكيل متخصص → 🔒 حوكمة: موافقة بشرية قبل الدفع/التوقيع/الإرسال الخارجي
```

خالد يستقبل الطلب، يملكه، يوزّعه على المتخصص المناسب، يجمع النتيجة، يوثّق، ويسلّم مخرجاً جاهزاً.

## مسار موازٍ — التوظيف (HR)

- **صاحب العمل:** `/join` → `/portal/join` (اشتراك) → `/dashboard` لوحة التوظيف / إدارة الوظائف والوصول لقاعدة المرشّحين.
- **الباحث عن عمل:** `/candidates` → `/portal/candidates` (الانضمام لقاعدة المرشّحين والتطابق مع الفرص).
